## Big-O Complexity

- O(1) [Excellent]
- O(log n) [Good]
- O(n) [Fair]
- O(n log n) [Bad]
- O(n^2) [Horrible]
- O(2^n) [Horribleeee]
- O(n^n) [Horribleeeeeee]

## Rules to calculate Big O notation

- Rule 1: Worst Case
- Rule 2: Remove Constants
- Rule 3: Different terms for inputs
- Rule 4: Drop non dominants
